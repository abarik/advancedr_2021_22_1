29 adding data label

ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_line() + geom_point() + geom_text(aes(label = round(pop, 2)), nudge_x = -3) + theme_classic()

ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_segment(aes(y = pop, x = year, yend = 0, xend = year)) + theme_classic()

Lollipop plot By combining the functions geom_segment() and geom_point(), we can produce a lollipop plot.

ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_segment(aes(yend = 0, xend = year)) + geom_point(aes(y = pop, x = year), size = 3) + theme_classic()

Area plot The function geom_area() is used to create an area plot.

ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_area() + theme_classic()

Step plot The function geom_step() is used to create a step plot with the argument direction indicating the direction of the plot.

ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_step(aes(y = pop, x = year)) + theme_classic() # vh (vertical then horizontal) ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_step(aes(y = pop, x = year), direction = ‘vh’) + theme_classic()

Line width The argument size, control line width.

ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_line(size = 1) + theme_classic()

Line style The argument linetype controls line style. It accepts the same values as base graphics that is, integers ranging from 0 to 6 and

‘blank’ = 0, ‘solid’ = 1 (default) ‘dashed’ = 2 ‘dotted’ = 3 ‘dotdash’ = 4 ‘longdash’ = 5 ‘twodash’ = 6 ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_line(size = 1, linetype = 2) + theme_classic()

ggplot(data = pop_growth, aes(y = pop, x = year)) + geom_line(size = 1, linetype = ‘twodash’) + theme_classic()

32.11 Summary functions Description ggplot() initializes a ggplot object qplot() creates ggplot plots quickly geom_point() creates a scatter plot geom_line() creates a line plot geom_area() creates an area plot geom_step() creates a step plot geom_dotplot() creates a dot plot geom_histogram() creates a histogram geom_density() creates a density plot geom_qq() and geom_qq_line() creates a q-q plot geom_boxplot() creates a box plot geom_violin() creates a violin plot geom_smooth() adds pattern to a plot geom_rug() adds rug to a plot geom_jitter() adds jitter to a plot geom_bar() and geom_col() creates bar charts geom_text() and geom_label() adds text and label to a plot stat_summary() computes and adds aggregations e.g. mean to a plot scale_shape_manual() sets mapped shapes scale_radius(), scale_size() and scale_size_area controls mapped sizes scale_colour_hue() and scale_fill_hue() default colour scale for categorical variables scale_colour_grey() defines grey colours for discrete variable scale_colour_manual() and scale_fill_manual() controls mapped colours manually scale_colour_gradient() and scale_fill_gradient() defines a two-colour gradient scale_colour_gradient2() and scale_fill_gradient2() defines a three-colour gradient (low-mid-high) scale_colour_gradientn() and scale_fill_gradientn() defines a more then three colour gradient scale_colour_brewer() and scale_fill_brewer() defines colour scale for discrete variables scale_colour_distiller() and scale_fill_distiller() defines colour scale for continuous variables scale_colour_viridis() and scale_fill_viridis() defines colour scale for discrete and continuous variables coord_cartesian() zooms a plot without changing the underlying data coord_fixed() controls the aspect ratio. coord_flip() flips a plot along its diagonal coord_polar() converts bar chart to pie chart facet_grid() and facet_wrap() creates subplots labs() modifies title, subtitle, and axis labels ggtitle() modifies title xlab() modifies x-axis label ylab() modifies y-axis label guides() controls legend ggsave() saves ggplot to disc theme() controls non-data elements
